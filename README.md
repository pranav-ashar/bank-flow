# 🏦 Bank Flow

![Python](https://img.shields.io/badge/Python-3.14-blue?logo=python)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)
![Level](https://img.shields.io/badge/Level-Beginner%20Project-purple)

A menu-driven ATM Banking System developed in Python that simulates essential banking operations through an interactive console interface. The project focuses on applying core programming concepts to solve a real-world problem in a structured and modular way.

---

## ✨ Features

- 🔐 Secure Login using Card Number & PIN
- 💰 Account Balance Inquiry
- 💵 Cash Deposit
- 💸 Cash Withdrawal
- 🔄 Money Transfer Between Accounts
- 📄 Mini Statement (Last 5 Transactions)
- 🔑 PIN Change
- 🔁 Multiple Transactions in a Single Session
- 🚪 Logout & Re-login Support

---

## 🛠 Concepts Used

- Conditional Statements
- Nested Dictionaries

- Menu-Driven Programming
- Data Validation

## 🎯 Project Highlights

- Designed using modular functions for better readability and maintainability.
- Transaction history is maintained separately for every customer.
- Implements minimum balance validation during withdrawals.
- Supports secure money transfers with automatic updates for both sender and receiver.
- Displays only the five most recent transactions using list slicing.
- Simulates the workflow of a real ATM in a console environment.
